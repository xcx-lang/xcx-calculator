# XCX Expression Calculator

A simple mathematical expression calculator written in XCX.

---

## Features

- Supports floating-point and integer numbers.
- Operators: `+`, `-`, `*`, `/`, `^`, `%`
- Mathematical constants: `PI`, `E`, `TAU`, `PHI`, `SQRT2`, `INF`
- Mathematical functions: `sqrt`, `cbrt`, `abs`, `floor`, `ceil`, `round`, `sin`, `cos`, `tan`, `asin`, `acos`, `atan`, `ln`, `log2`, `log10`, `exp`, `factorial`, `fibonacci`
- Supports expressions with parentheses.
- Handles negative numbers (unary minus).

---

## Installation

1. Clone the repository.
2. Ensure you have the XCX environment with the `math-lib` library installed.

---

## Usage

1. Run `main.xcx`.
2. Enter mathematical expressions in the console.
3. To exit, type `q` or `quit`.

### Examples

```text
2 + 3 * 4
= 14

sqrt(PI ^ 2) + sin(TAU / 4)
= 4.1415926535

factorial(10) / fibonacci(15)
= 5948.85
```
